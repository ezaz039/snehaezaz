GitHub Pages DNS Configuration Guide

Based on your DNS query, your domain is not properly configured for GitHub Pages.

1 Create CNAME File

In your GitHub repository, create a file named CNAME (all caps) with content:

ezaznitd.me

This tells GitHub Pages about your custom domain.

2 Configure DNS Records

In your domain registrar's DNS settings, create these records:

A
ezaznitd.me → 185.199.108.153
ezaznitd.me → 185.199.109.153
ezaznitd.me → 185.199.110.153
ezaznitd.me → 185.199.111.153
CNAME
www.ezaznitd.me → snehaezaz.github.io

3 Enable HTTPS

In your GitHub repository:

  • Go to Settings → Pages
  • Under "Custom domain", enter ezaznitd.me
  • Check "Enforce HTTPS"
  • Wait for DNS changes to propagate (up to 24 hours)

4 Verify Configuration

After making changes, verify with these commands:

# Check A records
dig +short A ezaznitd.me

# Check CNAME record
dig +short CNAME www.ezaznitd.me

# Check if domain points to GitHub
dig +short ezaznitd.me | grep -E "185\.199\.(108|109|110|111)\.153"
